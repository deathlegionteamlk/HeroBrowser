# HeroBrowser

HeroBrowser is a modern, fast, and secure web browser based on the open-source Chromium project (via Qt WebEngine). It is designed with a gaming-inspired UI and integrated AI features.

## Features
- **Chromium Engine**: Powered by the same engine that runs Google Chrome.
- **Opera GX Inspired UI**: Dark theme with neon accents and a custom sidebar.
- **Death Legion AI Assistant**: Built-in AI to help with your browsing.
- **Tab Management**: Efficiently manage multiple websites.
- **Android Support**: Project structure ready for Android deployment.

## Building from Source

### Prerequisites
- C++17 Compiler
- CMake 3.10+
- Qt 5.15+ with WebEngine module

### Build Instructions
```bash
mkdir build
cd build
cmake ..
make
./HeroBrowser
```
